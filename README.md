# NeuroAudit Investor SPA

Single-page React app that packages the NeuroAudit business model, investor memo, slide outline,
financial model summary, and AI governance playbooks into a shareable website.

## Run locally

```bash
npm install
npm run dev
```

Then open the printed localhost URL (default: http://localhost:5173).

## Deploy

1. Push this folder to GitHub.
2. In Vercel, create a new project from the repo.
3. Framework preset: Vite.
4. Build command: `npm run build`
5. Output directory: `dist`
